How it's made

The ideas behind the pixels

Every demo is a small, readable module. The interesting techniques in one place:

Vector fields from noise

Map each point in space to an angle via Perlin(x·s, y·s + t). Step particles along it and they trace smooth, organic currents.

Escape-time fractals

Iterate z → z² + c per pixel; how fast it escapes to infinity decides the color. Smooth shading removes the banding.

Emergent flocking

Three local rules — separation, alignment, cohesion — with no leader produce lifelike swarm motion.

Sound from math

The Web Audio API turns oscillators into notes; an AnalyserNode hands back the live FFT we paint as bars.

Newtonian gravity

Accumulate an inverse-square pull from each attractor, integrate velocity, and fade the canvas to leave orbital trails.

Perspective by hand

Project 3D stars to 2D with x' = x / z. Shrinking z each frame makes them rush toward the viewer.
